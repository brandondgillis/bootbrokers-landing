# Boot Brokers Landing

Landing page and blog for Boot Brokers marketplace.

## Deploy

This site is automatically deployed to Vercel.

## Local Development

```bash
npm install
npm run dev
```

## Structure

- `/` - Coming Soon page
- `/blog` - Blog home
- `/blog/[slug]` - Individual blog posts

## Adding Blog Posts

Edit `lib/posts.js` to add new blog posts.
